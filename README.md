# Приложение для приглашения на свадьбу

Современное веб-приложение для свадебного приглашения с интерактивными элементами и анимациями.

## Структура проекта

- `index.html` - основная HTML-страница
- `styles.css` - главный CSS файл, импортирующий модульные стили
- `src/` - директория с JavaScript модулями:
  - `main.js` - точка входа приложения
  - `utils.js` - общие утилиты
  - `hero.js` - модуль героя
  - `invite-date.js` - модуль приглашения и даты
  - `venue.js` - модуль площадки
  - `timeline.js` - модуль тайминга
  - `countdown.js` - модуль обратного отсчета
  - `wishes.js` - модуль пожеланий
  - `contacts.js` - модуль контактов
  - `guest-form.js` - модуль формы гостя
  - `finale.js` - модуль финальной секции
- `css/` - директория с модульными CSS файлами
- `assets/` - изображения и медиафайлы

## Особенности

- Модульная архитектура JavaScript
- Адаптивный дизайн
- Доступность (accessibility)
- Анимации и переходы
- Интерактивные элементы
- Поддержка современных браузеров

## Установка

Проект не требует установки зависимостей. Просто откройте `index.html` в браузере.
